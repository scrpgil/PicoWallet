# Developers #

### Build from source ###

1) Install npm modules:

```
git clone https://github.com/PicoWallet/PicoWallet.git
npm install
```

2) Install Ionic:

```
npm install -g ionic
```

3) Serve:

<pre>ionic serve</pre>

##### OR #####

4) Prod Build:

<pre>ionic build --prod</pre>
